# Stock Price Prediction Pipeline

Проект для прогнозирования цен акций с использованием ансамблевых методов машинного обучения.

## Описание

Этот проект представляет собой комплексный пайплайн для предсказания цен акций на основе исторических данных и технических индикаторов. Модель использует ансамбль различных алгоритмов машинного обучения для прогнозирования цен открытия, максимума, минимума, закрытия и объема торгов.


- **Предобработка данных**: Создание расширенных технических индикаторов и признаков
- **Ансамблевое моделирование**: Комбинация Random Forest, Gradient Boosting, Ridge и Extra Trees регрессоров
- **Многозадачное прогнозирование**: Одновременное предсказание нескольких целевых переменных
- **Оценка качества**: Комплексные метрики оценки включая MAE, RMSE, MAPE и специализированные конкурсные метрики
